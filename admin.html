How to run a scrape:
  1. Save your categories using the "Save & Download JSON" button above
  2. Commit the downloaded categories.json file to your GitHub repository
  3. Click "Open GitHub Actions"
  4. Click on "Scrape Prices and Build Site" workflow
  5. Click "Run workflow" button
  6. Optionally enter a category key (e.g., "Barbora:Beer")
  7. Click "Run workflow" to confirm
